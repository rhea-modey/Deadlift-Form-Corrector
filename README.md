# Real-Time Weightlifting Form Corrector (OpenCV + Mediapipe)

A real-time computer vision tool that detects improper deadlift form using
Mediapipe pose estimation and OpenCV. Provides instantaneous feedback to 
help athletes correct posture and prevent injuries.
